Berlin will eine neue Liste für <button type="button" class="article-term article-term--default" data-term-id="term-1">Mieten</button> machen. Das Parlament von Berlin soll <button type="button" class="article-term" data-term-id="term-2">darüber</button> <button type="button" class="article-term" data-term-id="term-3">entscheiden</button>. Die Liste <button type="button" class="article-term" data-term-id="term-4">betrifft</button> <button type="button" class="article-term" data-term-id="term-5">Wohnungen</button> in der Stadt Berlin.

Die Liste <button type="button" class="article-term" data-term-id="term-6">heißt</button> <button type="button" class="article-term article-term--default" data-term-id="term-7">Mietkataster</button>. In ihr stehen wichtige Daten zu <button type="button" class="article-term article-term--default" data-term-id="term-8">Mietwohnungen</button>. <button type="button" class="article-term" data-term-id="term-9">Dazu gehören</button> die Adresse, die <button type="button" class="article-term" data-term-id="term-10">Größe</button> und der <button type="button" class="article-term article-term--default" data-term-id="term-11">Preis pro Quadratmeter</button>. Das ist der Preis für ein kleines Stück Wohnfläche. <button type="button" class="article-term" data-term-id="term-12">Es geht um</button> <button type="button" class="article-term" data-term-id="term-13">etwa</button> 1,7 Millionen <button type="button" class="article-term" data-term-id="term-14">Verträge</button> für <button type="button" class="article-term article-term--default" data-term-id="term-8">Mietwohnungen</button>.

Ein <button type="button" class="article-term article-term--default" data-term-id="term-15">Amt</button> der <button type="button" class="article-term" data-term-id="term-16">Stadtregierung</button> soll die Daten zentral <button type="button" class="article-term" data-term-id="term-17">bekommen</button>. Die <button type="button" class="article-term" data-term-id="term-18">Mitarbeiter</button> können die Daten dann am Computer <button type="button" class="article-term" data-term-id="term-19">prüfen</button>. Auch Computerprogramme sollen helfen. Sie können viele Preise schnell <button type="button" class="article-term" data-term-id="term-20">vergleichen</button>. So sieht das <button type="button" class="article-term article-term--default" data-term-id="term-15">Amt</button> besser, wo <button type="button" class="article-term article-term--default" data-term-id="term-1">Mieten</button> sehr <button type="button" class="article-term" data-term-id="term-21">hoch</button> sind.

CDU und SPD <button type="button" class="article-term" data-term-id="term-22">regieren</button> Berlin <button type="button" class="article-term" data-term-id="term-23">zusammen</button> und planen die Liste. Die <button type="button" class="article-term" data-term-id="term-24">beiden</button> Parteien <button type="button" class="article-term" data-term-id="term-25">hoffen auf</button> mehr <button type="button" class="article-term" data-term-id="term-26">Klarheit</button>. Heute ist der <button type="button" class="article-term article-term--default" data-term-id="term-27">Mietmarkt</button> für viele Menschen <button type="button" class="article-term" data-term-id="term-28">schwer zu verstehen</button>. Viele <button type="button" class="article-term article-term--default" data-term-id="term-29">Mieter</button> wissen nicht, <button type="button" class="article-term" data-term-id="term-30">ob</button> ihr Preis <button type="button" class="article-term" data-term-id="term-31">richtig</button> ist.

So eine Liste <button type="button" class="article-term" data-term-id="term-32">gibt es</button> in Deutschland <button type="button" class="article-term" data-term-id="term-33">noch nicht</button>. Mit ihr kann die Stadt Regeln schneller <button type="button" class="article-term" data-term-id="term-19">prüfen</button>. Wenn <button type="button" class="article-term" data-term-id="term-34">jemand</button> <button type="button" class="article-term" data-term-id="term-35">zu viel</button> <button type="button" class="article-term" data-term-id="term-36">Miete</button> <button type="button" class="article-term" data-term-id="term-37">verlangt</button>, <button type="button" class="article-term" data-term-id="term-38">merkt</button> das <button type="button" class="article-term article-term--default" data-term-id="term-15">Amt</button> es <button type="button" class="article-term" data-term-id="term-39">früher</button>. Dann kann Berlin besser gegen zu hohe <button type="button" class="article-term article-term--default" data-term-id="term-1">Mieten</button> <button type="button" class="article-term" data-term-id="term-40">vorgehen</button>.
