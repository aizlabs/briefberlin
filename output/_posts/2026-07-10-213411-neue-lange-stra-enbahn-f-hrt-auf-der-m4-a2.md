Berlin hat eine neue sehr lange <button type="button" class="article-term article-term--default" data-term-id="term-1">Straßenbahn</button>. <button type="button" class="article-term" data-term-id="term-2">Seit</button> Mittwoch fährt der Urbanliner auf der Linie M4. Die <button type="button" class="article-term" data-term-id="term-3">Strecke</button> geht <button type="button" class="article-term" data-term-id="term-4">vom</button> Hackeschen Markt <button type="button" class="article-term" data-term-id="term-5">bis</button> zur Sulzfelder Straße.

Die BVG hat den Start <button type="button" class="article-term" data-term-id="term-6">leise</button> gemacht. <button type="button" class="article-term" data-term-id="term-7">Nächste Woche</button> <button type="button" class="article-term" data-term-id="term-8">zeigt</button> sie die Bahn Politikern und Medien. Viele <button type="button" class="article-term article-term--default" data-term-id="term-9">Fahrgäste</button> <button type="button" class="article-term" data-term-id="term-10">merken</button> die neue Bahn <button type="button" class="article-term" data-term-id="term-11">vielleicht</button> erst in <button type="button" class="article-term" data-term-id="term-12">engen Kurven</button>. <button type="button" class="article-term" data-term-id="term-13">Dort</button> sieht man ihre <button type="button" class="article-term" data-term-id="term-14">Länge</button> <button type="button" class="article-term" data-term-id="term-15">besonders</button> gut. Sie ist mehr als 50 Meter lang und hat neun Teile.

Innen gibt es viel mehr Platz als in <button type="button" class="article-term" data-term-id="term-16">älteren</button> Bahnen. 303 Menschen <button type="button" class="article-term" data-term-id="term-17">dürfen</button> <button type="button" class="article-term" data-term-id="term-18">mitfahren</button>. 88 Menschen können <button type="button" class="article-term" data-term-id="term-19">sitzen</button>, 215 Menschen können <button type="button" class="article-term" data-term-id="term-20">stehen</button>. Es gibt keine <button type="button" class="article-term article-term--default" data-term-id="term-21">Automaten</button> für <button type="button" class="article-term article-term--default" data-term-id="term-22">Fahrkarten</button> in der Bahn. Es gibt nur <button type="button" class="article-term" data-term-id="term-23">Geräte</button> zum <button type="button" class="article-term" data-term-id="term-24">Stempeln</button>. Die BVG sagt: <button type="button" class="article-term" data-term-id="term-25">Weniger</button> Menschen <button type="button" class="article-term" data-term-id="term-26">kaufen</button> <button type="button" class="article-term" data-term-id="term-27">Papierkarten</button> in der <button type="button" class="article-term article-term--default" data-term-id="term-1">Straßenbahn</button>. <button type="button" class="article-term" data-term-id="term-28">Deshalb</button> gibt es mehr <button type="button" class="article-term article-term--default" data-term-id="term-29">Verkaufsstellen</button> an der M4.

Der Start kam <button type="button" class="article-term" data-term-id="term-30">später als geplant</button>. Die erste Bahn kam schon im Juli 2024 nach Berlin. Aber die <button type="button" class="article-term article-term--default" data-term-id="term-31">Behörde</button> <button type="button" class="article-term" data-term-id="term-32">musste</button> <button type="button" class="article-term" data-term-id="term-33">zuerst</button> die <button type="button" class="article-term" data-term-id="term-34">Fahrt</button> mit Fahrgästen <button type="button" class="article-term" data-term-id="term-35">erlauben</button>. <button type="button" class="article-term" data-term-id="term-36">Außerdem</button> ist die Bahn sehr <button type="button" class="article-term" data-term-id="term-37">schwer</button>. Leer <button type="button" class="article-term" data-term-id="term-38">wiegt</button> sie etwa 63 Tonnen, <button type="button" class="article-term" data-term-id="term-39">voll</button> etwa 100 Tonnen. <button type="button" class="article-term" data-term-id="term-28">Deshalb</button> hat Berlin Tunnel am Alexanderplatz <button type="button" class="article-term" data-term-id="term-40">stärker gemacht</button>. Ende Juni sagte die <button type="button" class="article-term article-term--default" data-term-id="term-31">Behörde</button> Ja.

Noch darf die Bahn <button type="button" class="article-term" data-term-id="term-41">nicht überall</button> fahren. Der <button type="button" class="article-term" data-term-id="term-42">Fahrer</button> muss an einem <button type="button" class="article-term" data-term-id="term-43">bestimmten</button> Ende der Bahn <button type="button" class="article-term" data-term-id="term-19">sitzen</button>. Auf der M4 <button type="button" class="article-term" data-term-id="term-44">klappt</button> das, <button type="button" class="article-term" data-term-id="term-45">weil</button> es <button type="button" class="article-term" data-term-id="term-13">dort</button> <button type="button" class="article-term article-term--default" data-term-id="term-46">Schleifen</button> zum Wenden gibt. <button type="button" class="article-term" data-term-id="term-5">Bis</button> Ende des Jahres <button type="button" class="article-term" data-term-id="term-47">sollen</button> etwa 15 Urbanliner fahren. <button type="button" class="article-term" data-term-id="term-5">Bis</button> 2028 <button type="button" class="article-term" data-term-id="term-48">plant</button> die BVG 30 Bahnen auf der M4. <button type="button" class="article-term" data-term-id="term-49">Insgesamt</button> hat sie 65 <button type="button" class="article-term" data-term-id="term-50">Züge</button> <button type="button" class="article-term" data-term-id="term-51">bestellt</button>.

## Vokabeln

- **Straßenbahn** - tram / streetcar - Eine Bahn, die auf Schienen in der Stadt fährt.
- **Fahrgäste** - passengers - Menschen, die mit Bus oder Bahn fahren.
- **Automaten** - ticket machines / vending machines - Maschinen, an denen man etwas kaufen kann.
- **Fahrkarten** - tickets - Karten, die man für eine Fahrt braucht.
- **Verkaufsstellen** - sales points / places to buy tickets - Orte, an denen man etwas kaufen kann.
- **Behörde** - authority / government office - Eine offizielle Stelle vom Staat oder der Stadt.
- **Schleifen** - loops - Schienen, auf denen eine Bahn umdrehen kann.
