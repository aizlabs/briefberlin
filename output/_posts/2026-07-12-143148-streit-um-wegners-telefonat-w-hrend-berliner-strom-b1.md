Beim Berliner <button type="button" class="article-term article-term--default" data-term-id="term-1">Stromausfall</button> im Januar gibt es neue <button type="button" class="article-term article-term--default" data-term-id="term-4">Widersprüche</button>. Der <button type="button" class="article-term" data-term-id="term-2">Streit</button> <button type="button" class="article-term" data-term-id="term-5">betrifft</button> das Krisenmanagement von Kai Wegner, dem <button type="button" class="article-term" data-term-id="term-6">Regierenden Bürgermeister</button> von Berlin. Es <button type="button" class="article-term" data-term-id="term-7">geht um</button> die <button type="button" class="article-term" data-term-id="term-8">Frage</button>, <button type="button" class="article-term" data-term-id="term-9">ob</button> Wegner <button type="button" class="article-term" data-term-id="term-3">während</button> des <button type="button" class="article-term article-term--default" data-term-id="term-10">mehrtägigen</button> Blackouts mit Bundeskanzler Friedrich Merz <button type="button" class="article-term" data-term-id="term-11">telefoniert</button> hat. Ein Blackout ist ein großer <button type="button" class="article-term article-term--default" data-term-id="term-1">Stromausfall</button>, der länger <button type="button" class="article-term" data-term-id="term-12">dauern</button> kann. Für die Politik ist das Thema wichtig, weil es um Vertrauen in einer Krise geht.

Wegner hält an seiner <button type="button" class="article-term article-term--default" data-term-id="term-13">Darstellung</button> fest. Er sagt, dass er in dieser Zeit mit Merz gesprochen hat. Die Senatskanzlei unterstützt diese <button type="button" class="article-term" data-term-id="term-14">Aussage</button>. Diese <button type="button" class="article-term article-term--default" data-term-id="term-15">Behörde</button> arbeitet für den Berliner Senat und den Bürgermeister. Sie <button type="button" class="article-term" data-term-id="term-16">verweist</button> auf einen <button type="button" class="article-term" data-term-id="term-17">Eintrag</button> in Wegners Telefonliste. <button type="button" class="article-term" data-term-id="term-18">Nach ihren Angaben</button> <button type="button" class="article-term" data-term-id="term-19">fand</button> das <button type="button" class="article-term" data-term-id="term-20">Gespräch</button> am 4. Januar statt. Damit nennt die Berliner Seite ein <button type="button" class="article-term" data-term-id="term-21">konkretes</button> <button type="button" class="article-term" data-term-id="term-22">Datum</button> für das <button type="button" class="article-term" data-term-id="term-23">Telefonat</button>.

Aus dem <button type="button" class="article-term" data-term-id="term-26">Kanzleramt</button> kommt <button type="button" class="article-term" data-term-id="term-24">jedoch</button> eine andere und <button type="button" class="article-term" data-term-id="term-25">vorsichtigere</button> <button type="button" class="article-term article-term--default" data-term-id="term-13">Darstellung</button>. Das <button type="button" class="article-term" data-term-id="term-26">Kanzleramt</button> ist das Büro des Bundeskanzlers und <button type="button" class="article-term" data-term-id="term-27">prüft</button> solche Informationen. Nach den dort <button type="button" class="article-term" data-term-id="term-28">vorhandenen</button> Informationen <button type="button" class="article-term" data-term-id="term-29">lässt sich</button> ein solches <button type="button" class="article-term" data-term-id="term-23">Telefonat</button> nicht <button type="button" class="article-term article-term--default" data-term-id="term-30">feststellen</button>. Das bedeutet: In den <button type="button" class="article-term" data-term-id="term-31">Unterlagen</button> gibt es keinen <button type="button" class="article-term article-term--default" data-term-id="term-32">sicheren Beleg</button> dafür. <button type="button" class="article-term" data-term-id="term-33">Außerdem</button> <button type="button" class="article-term" data-term-id="term-34">erinnert</button> sich Merz selbst nicht an ein <button type="button" class="article-term" data-term-id="term-20">Gespräch</button> mit Wegner.

Der Widerspruch ist politisch <button type="button" class="article-term article-term--default" data-term-id="term-35">heikel</button>, <button type="button" class="article-term" data-term-id="term-36">obwohl</button> es nur um ein <button type="button" class="article-term" data-term-id="term-23">Telefonat</button> geht. Bei einem langen <button type="button" class="article-term article-term--default" data-term-id="term-1">Stromausfall</button> <button type="button" class="article-term" data-term-id="term-37">erwarten</button> viele Menschen schnelle <button type="button" class="article-term" data-term-id="term-38">Entscheidungen</button> und <button type="button" class="article-term" data-term-id="term-39">klare</button> Kommunikation. Der <button type="button" class="article-term" data-term-id="term-40">genaue</button> <button type="button" class="article-term" data-term-id="term-41">Ablauf</button> in solchen Tagen ist <button type="button" class="article-term" data-term-id="term-42">deshalb</button> wichtig. Er zeigt, wie die <button type="button" class="article-term" data-term-id="term-43">politische Spitze</button> Informationen <button type="button" class="article-term" data-term-id="term-44">sammelt</button> und <button type="button" class="article-term" data-term-id="term-38">Entscheidungen</button> <button type="button" class="article-term" data-term-id="term-45">vorbereitet</button>. Wenn <button type="button" class="article-term" data-term-id="term-46">unklar</button> bleibt, wer mit wem gesprochen hat, kann das <button type="button" class="article-term" data-term-id="term-47">Zweifel</button> <button type="button" class="article-term" data-term-id="term-48">auslösen</button>.

Aus einer Diskussion über Organisation wird <button type="button" class="article-term" data-term-id="term-42">deshalb</button> ein politisches <button type="button" class="article-term article-term--default" data-term-id="term-49">Glaubwürdigkeitsproblem</button>. Wegner muss erklären, warum seine <button type="button" class="article-term article-term--default" data-term-id="term-13">Darstellung</button> von der <button type="button" class="article-term article-term--default" data-term-id="term-13">Darstellung</button> aus dem <button type="button" class="article-term" data-term-id="term-26">Kanzleramt</button> <button type="button" class="article-term" data-term-id="term-50">abweicht</button>. Noch ist nicht <button type="button" class="article-term" data-term-id="term-51">bekannt</button>, <button type="button" class="article-term" data-term-id="term-9">ob</button> weitere Daten die <button type="button" class="article-term" data-term-id="term-8">Frage</button> <button type="button" class="article-term" data-term-id="term-52">klären</button> werden. Die Kontroverse <button type="button" class="article-term" data-term-id="term-53">dürfte</button> ihn <button type="button" class="article-term" data-term-id="term-42">deshalb</button> weiter <button type="button" class="article-term" data-term-id="term-54">beschäftigen</button>.
