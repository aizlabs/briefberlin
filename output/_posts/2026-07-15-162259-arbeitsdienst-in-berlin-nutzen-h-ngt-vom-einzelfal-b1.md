In Berlin gibt es <button type="button" class="article-term" data-term-id="term-3">erneut</button> eine Diskussion über Arbeit für Menschen, die <button type="button" class="article-term article-term--default" data-term-id="term-4">staatliche Leistungen</button> bekommen. Ein <button type="button" class="article-term" data-term-id="term-5">Vorschlag</button> <button type="button" class="article-term" data-term-id="term-6">sieht vor</button>, diese Menschen <button type="button" class="article-term" data-term-id="term-7">stärker</button> für <button type="button" class="article-term article-term--default" data-term-id="term-8">gemeinnützige Aufgaben</button> <button type="button" class="article-term" data-term-id="term-9">einzusetzen</button>. Dazu könnte zum Beispiel Arbeit für mehr <button type="button" class="article-term" data-term-id="term-10">Sauberkeit</button> in der Stadt gehören. Die Debatte <button type="button" class="article-term" data-term-id="term-11">betrifft</button> <button type="button" class="article-term" data-term-id="term-12">vor allem</button> <button type="button" class="article-term" data-term-id="term-13">Arbeitslose</button>, die Hilfe vom Jobcenter bekommen.

<button type="button" class="article-term" data-term-id="term-14">Im Mittelpunkt</button> stehen <button type="button" class="article-term" data-term-id="term-15">sogenannte</button> <button type="button" class="article-term article-term--default" data-term-id="term-16">Arbeitsgelegenheiten</button>. Das sind keine normalen Jobs mit einem regulären <button type="button" class="article-term" data-term-id="term-17">Arbeitsvertrag</button>. Jobcenter können solche <button type="button" class="article-term" data-term-id="term-18">Tätigkeiten</button> <button type="button" class="article-term" data-term-id="term-19">anbieten</button> oder <button type="button" class="article-term article-term--default" data-term-id="term-20">zuweisen</button>. Sie sollen Menschen helfen, wieder einen <button type="button" class="article-term article-term--default" data-term-id="term-21">festen Tagesablauf</button> zu finden. Außerdem können sie den Kontakt zur <button type="button" class="article-term" data-term-id="term-22">Arbeitswelt</button> <button type="button" class="article-term" data-term-id="term-23">erleichtern</button>.

<button type="button" class="article-term" data-term-id="term-24">Trotzdem</button> ist der <button type="button" class="article-term" data-term-id="term-1">Nutzen</button> nicht für alle gleich. Manche Menschen brauchen zuerst <button type="button" class="article-term" data-term-id="term-25">Beratung</button>, <button type="button" class="article-term article-term--default" data-term-id="term-26">Weiterbildung</button> oder Hilfe bei persönlichen Problemen. Für andere kann eine einfache <button type="button" class="article-term" data-term-id="term-27">Tätigkeit</button> ein erster Schritt zurück in Arbeit sein. Deshalb <button type="button" class="article-term" data-term-id="term-2">hängt</button> der <button type="button" class="article-term" data-term-id="term-28">Erfolg</button> stark vom einzelnen Fall ab. Eine <button type="button" class="article-term" data-term-id="term-29">Pflicht</button> für viele Menschen wäre nicht automatisch eine gute <button type="button" class="article-term" data-term-id="term-30">Lösung</button>.

In Berlin <button type="button" class="article-term" data-term-id="term-31">spielte</button> dieses Mittel zuletzt nur <button type="button" class="article-term" data-term-id="term-32">eine kleine Rolle</button>. Es gab <button type="button" class="article-term" data-term-id="term-33">etwa</button> 8.800 <button type="button" class="article-term article-term--default" data-term-id="term-16">Arbeitsgelegenheiten</button> für <button type="button" class="article-term" data-term-id="term-34">rund</button> 218.000 <button type="button" class="article-term" data-term-id="term-13">Arbeitslose</button>. Diese Zahlen zeigen einen großen <button type="button" class="article-term" data-term-id="term-35">Abstand</button> zwischen <button type="button" class="article-term" data-term-id="term-36">Angebot</button> und <button type="button" class="article-term article-term--default" data-term-id="term-37">Bedarf</button>. Sie zeigen auch, dass Jobcenter solche Aufgaben <button type="button" class="article-term" data-term-id="term-38">bisher</button> <button type="button" class="article-term" data-term-id="term-39">eher</button> <button type="button" class="article-term" data-term-id="term-40">gezielt</button> <button type="button" class="article-term" data-term-id="term-41">genutzt</button> haben. Sie haben sie also nicht für alle Arbeitslosen <button type="button" class="article-term" data-term-id="term-42">in gleicher Weise</button> eingesetzt. Von einem <button type="button" class="article-term" data-term-id="term-43">Einsatz</button> in der ganzen Stadt kann man deshalb nicht sprechen. <button type="button" class="article-term" data-term-id="term-38">Bisher</button> <button type="button" class="article-term" data-term-id="term-44">ging es</button> <button type="button" class="article-term" data-term-id="term-39">eher</button> um einzelne Gruppen und konkrete Plätze.

Die neue Diskussion <button type="button" class="article-term" data-term-id="term-45">knüpft an</button> eine ältere Frage der <button type="button" class="article-term" data-term-id="term-46">Arbeitsmarktpolitik</button> an. Soll der Staat Menschen zu Arbeit <button type="button" class="article-term article-term--default" data-term-id="term-47">verpflichten</button>, wenn sie Leistungen <button type="button" class="article-term" data-term-id="term-48">erhalten</button>? Oder soll er <button type="button" class="article-term" data-term-id="term-7">stärker</button> auf <button type="button" class="article-term" data-term-id="term-49">freiwillige Hilfe</button> und <button type="button" class="article-term" data-term-id="term-50">passende Förderung</button> setzen? Die Antwort ist schwierig, weil die <button type="button" class="article-term" data-term-id="term-51">Lebenslagen</button> sehr verschieden sind. Klar ist nur: Ein <button type="button" class="article-term" data-term-id="term-43">Einsatz</button> für die Stadt kann <button type="button" class="article-term" data-term-id="term-52">sinnvoll</button> sein, wenn er gut <button type="button" class="article-term" data-term-id="term-53">passt</button> und fair organisiert ist.
