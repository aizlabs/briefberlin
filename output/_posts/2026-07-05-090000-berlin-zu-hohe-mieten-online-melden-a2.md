In Berlin können <button type="button" class="article-term article-term--default" data-term-id="term-4">Mieterinnen und Mieter</button> <button type="button" class="article-term" data-term-id="term-5">ab jetzt</button> <button type="button" class="article-term" data-term-id="term-1">zu hohe</button> <button type="button" class="article-term article-term--default" data-term-id="term-2">Mieten</button> online <button type="button" class="article-term article-term--default" data-term-id="term-3">melden</button>. <button type="button" class="article-term" data-term-id="term-6">Das geht</button> <button type="button" class="article-term" data-term-id="term-7">über</button> die Internet-Seite der Stadt für <button type="button" class="article-term" data-term-id="term-8">Dienste</button>. Die <button type="button" class="article-term article-term--default" data-term-id="term-9">Meldung</button> <button type="button" class="article-term" data-term-id="term-10">kommt</button> dann beim <button type="button" class="article-term article-term--default" data-term-id="term-11">Bezirksamt</button> an.

Das <button type="button" class="article-term article-term--default" data-term-id="term-11">Bezirksamt</button> ist das Amt für einen <button type="button" class="article-term" data-term-id="term-12">Teil</button> der Stadt. Die Stadt hat <button type="button" class="article-term" data-term-id="term-13">dort</button> eine neue Funktion im Internet. <button type="button" class="article-term" data-term-id="term-14">Zuerst</button> <button type="button" class="article-term" data-term-id="term-15">füllen</button> die Menschen ein <button type="button" class="article-term" data-term-id="term-16">Formular</button> aus. Dann <button type="button" class="article-term" data-term-id="term-17">schicken</button> sie die <button type="button" class="article-term article-term--default" data-term-id="term-9">Meldung</button> an ihr <button type="button" class="article-term article-term--default" data-term-id="term-11">Bezirksamt</button>. So können die Ämter die <button type="button" class="article-term" data-term-id="term-18">Hinweise</button> besser <button type="button" class="article-term" data-term-id="term-19">sammeln</button>.

Das <button type="button" class="article-term" data-term-id="term-20">Angebot</button> ist für Menschen in einer <button type="button" class="article-term article-term--default" data-term-id="term-21">Wohnung</button>. Sie <button type="button" class="article-term" data-term-id="term-22">glauben</button> <button type="button" class="article-term" data-term-id="term-23">vielleicht</button>: Unsere Miete ist <button type="button" class="article-term" data-term-id="term-24">viel zu hoch</button>. Nach dem <button type="button" class="article-term article-term--default" data-term-id="term-25">Gesetz</button> <button type="button" class="article-term" data-term-id="term-26">darf</button> eine Miete nur <button type="button" class="article-term" data-term-id="term-27">bis zu</button> einer <button type="button" class="article-term article-term--default" data-term-id="term-28">Grenze</button> <button type="button" class="article-term" data-term-id="term-29">steigen</button>. Diese <button type="button" class="article-term article-term--default" data-term-id="term-28">Grenze</button> <button type="button" class="article-term" data-term-id="term-30">schützt</button> <button type="button" class="article-term article-term--default" data-term-id="term-4">Mieterinnen und Mieter</button> in Berlin.

Nach der <button type="button" class="article-term article-term--default" data-term-id="term-9">Meldung</button> <button type="button" class="article-term" data-term-id="term-31">prüft</button> das Amt den <button type="button" class="article-term" data-term-id="term-32">Fall</button>. <button type="button" class="article-term" data-term-id="term-33">Danach</button> <button type="button" class="article-term" data-term-id="term-34">nimmt</button> das Amt Kontakt mit den Menschen auf. Es fragt <button type="button" class="article-term" data-term-id="term-23">vielleicht</button> nach <button type="button" class="article-term" data-term-id="term-35">weiteren</button> <button type="button" class="article-term" data-term-id="term-36">Papieren</button> oder Informationen. <button type="button" class="article-term" data-term-id="term-37">Am Ende</button> <button type="button" class="article-term" data-term-id="term-38">erklärt</button> das Amt die <button type="button" class="article-term" data-term-id="term-39">nächsten Schritte</button>.
