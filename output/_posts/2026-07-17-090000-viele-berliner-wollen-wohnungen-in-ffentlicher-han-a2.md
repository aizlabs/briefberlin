In Berlin gibt es neue Zahlen zu <button type="button" class="article-term" data-term-id="term-3">großen</button> <button type="button" class="article-term article-term--default" data-term-id="term-2">Wohnungsfirmen</button>. Viele Menschen wollen: Die Stadt soll <button type="button" class="article-term" data-term-id="term-1">Wohnungen</button> von <button type="button" class="article-term" data-term-id="term-3">großen</button> privaten Firmen <button type="button" class="article-term article-term--default" data-term-id="term-4">übernehmen</button>. 1021 Berliner mit <button type="button" class="article-term article-term--default" data-term-id="term-5">Wahlrecht</button> <button type="button" class="article-term" data-term-id="term-6">nahmen</button> im Februar an einer <button type="button" class="article-term article-term--default" data-term-id="term-7">Befragung</button> teil.

Im Jahr 2021 gab es schon einen <button type="button" class="article-term article-term--default" data-term-id="term-12">Volksentscheid</button>. Das ist eine <button type="button" class="article-term" data-term-id="term-13">Abstimmung</button> der <button type="button" class="article-term" data-term-id="term-14">Bürger</button>. <button type="button" class="article-term" data-term-id="term-15">Damals</button> sagten 59,1 Prozent Ja.

Das Thema ist politisch <button type="button" class="article-term" data-term-id="term-16">wichtig</button>. Der Senat ist die <button type="button" class="article-term" data-term-id="term-17">Regierung</button> von Berlin. Die <button type="button" class="article-term" data-term-id="term-13">Abstimmung</button> <button type="button" class="article-term" data-term-id="term-18">machte Druck</button> auf den Senat. Viele Berliner sehen <button type="button" class="article-term" data-term-id="term-25">Mieten</button> als wichtiges Problem. 46 Prozent <button type="button" class="article-term" data-term-id="term-26">nennen</button> <button type="button" class="article-term" data-term-id="term-27">Wohnen</button> und <button type="button" class="article-term" data-term-id="term-25">Mieten</button> als <button type="button" class="article-term" data-term-id="term-28">wichtigstes</button> Thema für die <button type="button" class="article-term" data-term-id="term-29">nächste Wahl</button>. Das Thema <button type="button" class="article-term" data-term-id="term-30">steht vor</button> Sicherheit und <button type="button" class="article-term" data-term-id="term-31">Einwanderung</button>. Befürworter <button type="button" class="article-term" data-term-id="term-32">hoffen auf</button> mehr <button type="button" class="article-term" data-term-id="term-33">Schutz</button> für <button type="button" class="article-term" data-term-id="term-34">Mieter</button>. Sie sagen auch: <button type="button" class="article-term" data-term-id="term-1">Wohnungen</button> sollen kein <button type="button" class="article-term" data-term-id="term-36">Geschäft</button> für hohe <button type="button" class="article-term" data-term-id="term-37">Gewinne</button> sein.

Aber viele Menschen haben <button type="button" class="article-term" data-term-id="term-38">Sorgen</button>. <button type="button" class="article-term" data-term-id="term-39">Jeder zweite</button> <button type="button" class="article-term" data-term-id="term-40">glaubt</button>: Die Stadt kann so viele <button type="button" class="article-term" data-term-id="term-1">Wohnungen</button> nicht gut <button type="button" class="article-term article-term--default" data-term-id="term-41">verwalten</button>. Andere <button type="button" class="article-term" data-term-id="term-42">fürchten</button> weniger neue <button type="button" class="article-term" data-term-id="term-1">Wohnungen</button> von privaten Firmen. Sie <button type="button" class="article-term" data-term-id="term-42">fürchten</button> auch hohe <button type="button" class="article-term" data-term-id="term-43">Kosten</button> für Berlin und die <button type="button" class="article-term article-term--default" data-term-id="term-44">Steuerzahler</button>. Gegner sind oft <button type="button" class="article-term" data-term-id="term-45">älter</button> und haben mehr Geld. Befürworter sind oft jünger und <button type="button" class="article-term" data-term-id="term-46">spüren</button> hohe <button type="button" class="article-term" data-term-id="term-25">Mieten</button> <button type="button" class="article-term" data-term-id="term-47">stärker</button>. <button type="button" class="article-term" data-term-id="term-48">Am Ende</button> können die unsicheren Menschen sehr <button type="button" class="article-term" data-term-id="term-16">wichtig</button> <button type="button" class="article-term" data-term-id="term-49">werden</button>.
