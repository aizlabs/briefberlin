In mehreren <button type="button" class="article-term article-term--default" data-term-id="term-2">Außenbezirken</button> von Berlin gibt es <button type="button" class="article-term" data-term-id="term-1">Ärger</button>. <button type="button" class="article-term" data-term-id="term-3">Seit Wochen</button> <button type="button" class="article-term" data-term-id="term-4">stehen</button> dort viele Bootsanhänger am <button type="button" class="article-term article-term--default" data-term-id="term-5">Straßenrand</button>. Mit solchen <button type="button" class="article-term" data-term-id="term-6">Anhängern</button> bringen Menschen <button type="button" class="article-term" data-term-id="term-7">Boote</button> zum Wasser.

<button type="button" class="article-term" data-term-id="term-8">Besonders oft</button> passiert das in Wannsee, Zehlendorf, Reinickendorf und Grunewald. Die Anhänger <button type="button" class="article-term" data-term-id="term-4">stehen</button> an großen Straßen und <button type="button" class="article-term" data-term-id="term-9">nahe</button> am Wasser. Auch Bewohner, Besucher und <button type="button" class="article-term article-term--default" data-term-id="term-10">Pendler</button> brauchen die <button type="button" class="article-term" data-term-id="term-11">Parkplätze</button>. <button type="button" class="article-term article-term--default" data-term-id="term-10">Pendler</button> fahren von dort oft mit der <button type="button" class="article-term" data-term-id="term-12">Bahn</button> <button type="button" class="article-term" data-term-id="term-13">weiter</button>. Die Regel ist klar: <button type="button" class="article-term" data-term-id="term-14">Ohne Auto</button> <button type="button" class="article-term" data-term-id="term-15">darf</button> ein Anhänger <button type="button" class="article-term article-term--default" data-term-id="term-16">höchstens</button> 14 Tage dort <button type="button" class="article-term" data-term-id="term-4">stehen</button>.

Die Kontrolle dieser Regel ist aber sehr <button type="button" class="article-term" data-term-id="term-17">schwer</button>. Wenn ein Anhänger <button type="button" class="article-term" data-term-id="term-18">länger</button> steht, <button type="button" class="article-term" data-term-id="term-19">zahlt</button> der <button type="button" class="article-term" data-term-id="term-20">Besitzer</button> nur 20 Euro. <button type="button" class="article-term" data-term-id="term-21">Deshalb</button> <button type="button" class="article-term" data-term-id="term-22">mieten</button> manche <button type="button" class="article-term" data-term-id="term-20">Besitzer</button> keinen privaten Platz für den Anhänger. Solche Plätze sind oft teuer, und es gibt <button type="button" class="article-term" data-term-id="term-23">zu wenige</button>.

Die <button type="button" class="article-term article-term--default" data-term-id="term-24">Ordnungsämter</button> kontrollieren die Regel auf den Straßen. Diese <button type="button" class="article-term" data-term-id="term-25">Ämter</button> fotografieren <button type="button" class="article-term" data-term-id="term-26">zum Beispiel</button> Ventile an den <button type="button" class="article-term" data-term-id="term-27">Reifen</button>. <button type="button" class="article-term" data-term-id="term-28">Später</button> <button type="button" class="article-term" data-term-id="term-29">prüfen</button> sie, ob der <button type="button" class="article-term" data-term-id="term-20">Besitzer</button> den Anhänger <button type="button" class="article-term" data-term-id="term-30">bewegt</button> hat.

Manchmal können die <button type="button" class="article-term article-term--default" data-term-id="term-31">Behörden</button> <button type="button" class="article-term" data-term-id="term-32">trotzdem</button> leider <button type="button" class="article-term" data-term-id="term-33">nichts tun</button>. <button type="button" class="article-term" data-term-id="term-34">Das gilt</button>, wenn <button type="button" class="article-term" data-term-id="term-20">Besitzer</button> ihre Anhänger regelmäßig bewegen. Es gilt auch, wenn die <button type="button" class="article-term" data-term-id="term-20">Besitzer</button> alles richtig <button type="button" class="article-term" data-term-id="term-35">anmelden</button> und parken. Niemand hat ein <button type="button" class="article-term" data-term-id="term-36">Recht</button> auf einen Parkplatz vor der <button type="button" class="article-term" data-term-id="term-37">eigenen Wohnung</button>. <button type="button" class="article-term" data-term-id="term-32">Trotzdem</button> <button type="button" class="article-term" data-term-id="term-38">bleibt</button> das Thema für die Politik wichtig.

<button type="button" class="article-term" data-term-id="term-39">Vereine</button> für Wassersport sehen ihre <button type="button" class="article-term" data-term-id="term-40">Mitglieder</button> nicht als Hauptproblem. Sie sagen: Viele <button type="button" class="article-term" data-term-id="term-39">Vereine</button> haben eigene Plätze für solche Anhänger. Sie sehen das Problem <button type="button" class="article-term" data-term-id="term-41">eher</button> bei <button type="button" class="article-term" data-term-id="term-42">Bootsbesitzern</button> ohne Verein.

Die Stadt Berlin <button type="button" class="article-term" data-term-id="term-43">sucht jetzt nach</button> einfachen <button type="button" class="article-term" data-term-id="term-44">Lösungen</button>. In Lichterfelde <button type="button" class="article-term" data-term-id="term-45">dürfen</button> auf einigen Plätzen nur Autos parken. Eine <button type="button" class="article-term" data-term-id="term-46">ähnliche</button> Regel soll am S-Bahnhof Wannsee kommen. So <button type="button" class="article-term" data-term-id="term-47">entstehen</button> dort wieder mehr <button type="button" class="article-term" data-term-id="term-11">Parkplätze</button> für <button type="button" class="article-term article-term--default" data-term-id="term-10">Pendler</button>. Aber private Plätze für Bootsanhänger bleiben <button type="button" class="article-term" data-term-id="term-13">weiter</button> <button type="button" class="article-term article-term--default" data-term-id="term-48">knapp</button>.
