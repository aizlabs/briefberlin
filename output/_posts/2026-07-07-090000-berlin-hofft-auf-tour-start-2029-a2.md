Berlin kann 2029 <button type="button" class="article-term" data-term-id="term-1">vielleicht</button> den Start der Tour de France haben. Die Tour de France ist ein sehr großes <button type="button" class="article-term article-term--default" data-term-id="term-2">Radrennen</button>. Eine deutsche <button type="button" class="article-term" data-term-id="term-5">Gruppe</button> hat <button type="button" class="article-term" data-term-id="term-3">nun</button> <button type="button" class="article-term" data-term-id="term-4">Interesse angemeldet</button>.

Die <button type="button" class="article-term" data-term-id="term-5">Gruppe</button> <button type="button" class="article-term" data-term-id="term-6">heißt</button> Grand Départ Allemagne. Sie <button type="button" class="article-term" data-term-id="term-7">arbeitet</button> mit dem deutschen <button type="button" class="article-term article-term--default" data-term-id="term-8">Radsportverband</button> <button type="button" class="article-term" data-term-id="term-9">zusammen</button>. Auch die <button type="button" class="article-term article-term--default" data-term-id="term-10">Sportverwaltung</button> von Berlin <button type="button" class="article-term" data-term-id="term-11">weiß davon</button>. Das ist die <button type="button" class="article-term" data-term-id="term-12">Stelle</button> in der Stadt, die für Sport <button type="button" class="article-term article-term--default" data-term-id="term-13">zuständig</button> ist. <button type="button" class="article-term" data-term-id="term-9">Zusammen</button> haben sie mit ASO <button type="button" class="article-term" data-term-id="term-14">gesprochen</button>.

ASO <button type="button" class="article-term" data-term-id="term-15">organisiert</button> die Tour de France. Die <button type="button" class="article-term" data-term-id="term-5">Gruppe</button> sagt: Berlin möchte den ersten Teil des <button type="button" class="article-term" data-term-id="term-16">Rennens</button> machen. Dieser Start <button type="button" class="article-term" data-term-id="term-6">heißt</button> Grand Départ. Das <button type="button" class="article-term" data-term-id="term-17">bedeutet</button>: Die Tour <button type="button" class="article-term" data-term-id="term-18">beginnt</button> dort. <button type="button" class="article-term" data-term-id="term-19">Noch</button> ist aber nichts <button type="button" class="article-term" data-term-id="term-20">sicher</button>.

<button type="button" class="article-term" data-term-id="term-21">Zuerst</button> <button type="button" class="article-term" data-term-id="term-22">soll es</button> ein <button type="button" class="article-term" data-term-id="term-23">kurzes</button> <button type="button" class="article-term article-term--default" data-term-id="term-24">Einzelzeitfahren</button> geben. <button type="button" class="article-term" data-term-id="term-25">Dabei</button> fährt jeder <button type="button" class="article-term" data-term-id="term-26">Fahrer</button> <button type="button" class="article-term" data-term-id="term-27">allein</button> <button type="button" class="article-term article-term--default" data-term-id="term-28">gegen die Uhr</button>. <button type="button" class="article-term" data-term-id="term-29">Danach</button> kann die Tour in anderen Städten <button type="button" class="article-term" data-term-id="term-30">weitergehen</button>. So ein Start <button type="button" class="article-term" data-term-id="term-31">bringt</button> viele <button type="button" class="article-term" data-term-id="term-32">Besucher</button> in die Stadt.

Als <button type="button" class="article-term" data-term-id="term-33">Ziel</button> für diesen ersten Teil ist das Brandenburger Tor <button type="button" class="article-term article-term--default" data-term-id="term-34">im Gespräch</button>. Es <button type="button" class="article-term" data-term-id="term-35">steht</button> am Pariser Platz in Berlin. Dort ist auch die französische <button type="button" class="article-term" data-term-id="term-36">Botschaft</button> <button type="button" class="article-term" data-term-id="term-37">in der Nähe</button>. <button type="button" class="article-term" data-term-id="term-38">Deshalb</button> <button type="button" class="article-term" data-term-id="term-39">passt</button> der <button type="button" class="article-term" data-term-id="term-40">Ort</button> gut zur Tour de France.
