Die <button type="button" class="article-term article-term--default" data-term-id="term-1">Wohnungssuche</button> in deutschen <button type="button" class="article-term" data-term-id="term-3">Großstädten</button> <button type="button" class="article-term" data-term-id="term-4">wird</button> für viele Menschen immer <button type="button" class="article-term" data-term-id="term-5">schwieriger</button>. Wer jetzt eine neue Wohnung <button type="button" class="article-term" data-term-id="term-6">sucht</button>, muss oft <button type="button" class="article-term" data-term-id="term-7">deutlich</button> höhere <button type="button" class="article-term article-term--default" data-term-id="term-2">Mieten</button> <button type="button" class="article-term" data-term-id="term-8">einplanen</button>. Das <button type="button" class="article-term" data-term-id="term-9">betrifft</button> besonders <button type="button" class="article-term article-term--default" data-term-id="term-10">Haushalte</button>, also einzelne Personen, Paare oder Familien mit gemeinsamem <button type="button" class="article-term article-term--default" data-term-id="term-11">Einkommen</button>.

Der <button type="button" class="article-term" data-term-id="term-12">Grund</button> ist der starke <button type="button" class="article-term" data-term-id="term-13">Unterschied</button> zwischen <button type="button" class="article-term article-term--default" data-term-id="term-2">Mieten</button> und Löhnen. In den <button type="button" class="article-term" data-term-id="term-14">vergangenen</button> drei Jahren sind die <button type="button" class="article-term article-term--default" data-term-id="term-15">Reallöhne</button> nur um 5,1 Prozent <button type="button" class="article-term" data-term-id="term-16">gestiegen</button>. <button type="button" class="article-term article-term--default" data-term-id="term-15">Reallöhne</button> zeigen, wie viel vom <button type="button" class="article-term" data-term-id="term-17">Lohn</button> nach höheren Preisen noch <button type="button" class="article-term" data-term-id="term-18">übrig bleibt</button>. <button type="button" class="article-term" data-term-id="term-19">Gleichzeitig</button> stiegen die <button type="button" class="article-term article-term--default" data-term-id="term-20">Angebotsmieten</button> in den größten Städten <button type="button" class="article-term" data-term-id="term-21">teilweise</button> um <button type="button" class="article-term" data-term-id="term-22">rund</button> <button type="button" class="article-term" data-term-id="term-23">ein Fünftel</button>. <button type="button" class="article-term article-term--default" data-term-id="term-20">Angebotsmieten</button> sind die Preise, die <button type="button" class="article-term article-term--default" data-term-id="term-24">Vermieter</button> bei neuen Wohnungen <button type="button" class="article-term" data-term-id="term-25">verlangen</button>.

Damit <button type="button" class="article-term" data-term-id="term-26">wachsen</button> die <button type="button" class="article-term" data-term-id="term-27">Kosten</button> für neue <button type="button" class="article-term article-term--default" data-term-id="term-28">Mietverträge</button> viel schneller als die <button type="button" class="article-term article-term--default" data-term-id="term-11">Einkommen</button>. Für viele <button type="button" class="article-term" data-term-id="term-29">Suchende</button> <button type="button" class="article-term" data-term-id="term-30">bedeutet</button> das eine schwere finanzielle <button type="button" class="article-term" data-term-id="term-31">Belastung</button>. <button type="button" class="article-term" data-term-id="term-32">Obwohl</button> sie arbeiten und mehr Geld <button type="button" class="article-term" data-term-id="term-33">verdienen</button> als <button type="button" class="article-term" data-term-id="term-34">früher</button>, reicht das <button type="button" class="article-term article-term--default" data-term-id="term-11">Einkommen</button> oft <button type="button" class="article-term" data-term-id="term-35">weniger weit</button>. Besonders problematisch ist ein <button type="button" class="article-term" data-term-id="term-36">Umzug</button>, denn dann <button type="button" class="article-term" data-term-id="term-37">gilt</button> meist ein neuer <button type="button" class="article-term" data-term-id="term-38">Vertrag</button> mit höherer Miete.

Die <button type="button" class="article-term" data-term-id="term-39">Entwicklung</button> zeigt, dass <button type="button" class="article-term" data-term-id="term-40">Wohnen</button> in großen Städten weniger bezahlbar <button type="button" class="article-term" data-term-id="term-4">wird</button>. Bezahlbar heißt hier: Die Miete <button type="button" class="article-term" data-term-id="term-41">passt</button> noch zum <button type="button" class="article-term" data-term-id="term-42">monatlichen</button> <button type="button" class="article-term article-term--default" data-term-id="term-11">Einkommen</button>. Wenn die Miete <button type="button" class="article-term" data-term-id="term-43">stärker</button> steigt als der <button type="button" class="article-term" data-term-id="term-17">Lohn</button>, bleibt weniger Geld für andere <button type="button" class="article-term" data-term-id="term-44">Ausgaben</button>. Dazu <button type="button" class="article-term" data-term-id="term-45">gehören</button> <button type="button" class="article-term" data-term-id="term-46">Lebensmittel</button>, Energie, Verkehr, <button type="button" class="article-term" data-term-id="term-47">Kleidung</button> und <button type="button" class="article-term" data-term-id="term-48">Freizeit</button>. Wichtig ist: Nicht jede Stadt und nicht jede Wohnung ist <button type="button" class="article-term" data-term-id="term-49">gleich teuer</button> geworden. Die <button type="button" class="article-term" data-term-id="term-50">Zahl</button> von <button type="button" class="article-term" data-term-id="term-22">rund</button> einem Fünftel <button type="button" class="article-term" data-term-id="term-51">beschreibt</button> besonders starke <button type="button" class="article-term" data-term-id="term-52">Anstiege</button> in den größten Städten.

Der <button type="button" class="article-term article-term--default" data-term-id="term-53">Wohnungsmarkt</button> <button type="button" class="article-term" data-term-id="term-54">entfernt sich</button> <button type="button" class="article-term" data-term-id="term-55">deshalb</button> weiter von dem, was viele Menschen aus ihrem normalen <button type="button" class="article-term article-term--default" data-term-id="term-11">Einkommen</button> <button type="button" class="article-term" data-term-id="term-56">bezahlen</button> können. Das Problem <button type="button" class="article-term" data-term-id="term-9">betrifft</button> nicht nur Menschen mit sehr wenig Geld. Auch <button type="button" class="article-term" data-term-id="term-57">breite Teile</button> der <button type="button" class="article-term" data-term-id="term-58">Bevölkerung</button> <button type="button" class="article-term" data-term-id="term-59">spüren</button> den <button type="button" class="article-term" data-term-id="term-60">Druck</button>, wenn sie eine neue Wohnung brauchen. <button type="button" class="article-term" data-term-id="term-61">Trotzdem</button> bleibt die <button type="button" class="article-term" data-term-id="term-62">Nachfrage</button> in <button type="button" class="article-term" data-term-id="term-3">Großstädten</button> hoch, weil dort viele <button type="button" class="article-term" data-term-id="term-63">Arbeitsplätze</button>, <button type="button" class="article-term" data-term-id="term-64">Hochschulen</button> und Angebote liegen.
