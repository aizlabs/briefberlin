Berlin will als erstes <button type="button" class="article-term" data-term-id="term-2">Bundesland</button> ein <button type="button" class="article-term article-term--default" data-term-id="term-1">Mietenkataster</button> <button type="button" class="article-term" data-term-id="term-3">einführen</button>, also ein <button type="button" class="article-term" data-term-id="term-4">zentrales</button> Register für <button type="button" class="article-term" data-term-id="term-5">Mietdaten</button>. Das Abgeordnetenhaus, das Berliner Parlament, soll den Plan am Donnerstag <button type="button" class="article-term" data-term-id="term-6">beschließen</button>. Ab dem kommenden Jahr sollen zu etwa 1,7 Millionen <button type="button" class="article-term" data-term-id="term-7">Mietverträgen</button> wichtige <button type="button" class="article-term" data-term-id="term-8">Angaben</button> <button type="button" class="article-term" data-term-id="term-9">gemeldet</button> werden. Dazu gehören die <button type="button" class="article-term" data-term-id="term-10">Wohnungsgröße</button>, die Adresse und der Preis pro Quadratmeter. Die <button type="button" class="article-term" data-term-id="term-11">zuständige</button> <button type="button" class="article-term" data-term-id="term-12">Senatsverwaltung</button> soll die Daten digital <button type="button" class="article-term" data-term-id="term-13">prüfen</button> können. Auch <button type="button" class="article-term" data-term-id="term-14">künstliche Intelligenz</button>, also lernende Computerprogramme, kann dabei helfen. Die Regierung aus CDU und SPD hofft auf mehr Kontrolle bei möglichen <button type="button" class="article-term article-term--default" data-term-id="term-15">Verstößen</button> gegen das <button type="button" class="article-term article-term--default" data-term-id="term-16">Mietrecht</button>.

Die <button type="button" class="article-term" data-term-id="term-17">Erwartungen</button> sind groß, aber es gibt <button type="button" class="article-term" data-term-id="term-18">Streit</button>. <button type="button" class="article-term" data-term-id="term-19">Unterstützer</button> sehen das Register als Hilfe gegen zu hohe Mieten. Bisher müssen meist einzelne Mieter selbst aktiv werden, wenn sie einen <button type="button" class="article-term" data-term-id="term-20">Verdacht</button> haben. Politiker im Land <button type="button" class="article-term" data-term-id="term-21">vermuten</button>, dass viele Berliner Verträge über den <button type="button" class="article-term" data-term-id="term-22">erlaubten Grenzen</button> liegen könnten. Besonders wichtig ist dabei die Mietpreisbremse. Diese Regel soll in <button type="button" class="article-term" data-term-id="term-23">angespannten</button> Städten starke <button type="button" class="article-term" data-term-id="term-24">Mietsteigerungen</button> <button type="button" class="article-term" data-term-id="term-25">begrenzen</button>. Das neue Register könnte den <button type="button" class="article-term" data-term-id="term-26">Behörden</button> zeigen, wo Mieten deutlich über <button type="button" class="article-term" data-term-id="term-27">vergleichbaren</button> Wohnungen liegen. Außerdem könnten Vermieter schon wegen möglicher Kontrollen zu hohe <button type="button" class="article-term" data-term-id="term-28">Forderungen</button> <button type="button" class="article-term" data-term-id="term-29">senken</button>.

<button type="button" class="article-term" data-term-id="term-30">Verbände</button> der <button type="button" class="article-term" data-term-id="term-31">Immobilienwirtschaft</button> kritisieren den Plan deutlich. Sie <button type="button" class="article-term" data-term-id="term-32">warnen vor</button> einem zu starken <button type="button" class="article-term article-term--default" data-term-id="term-33">Eingriff</button> in die Rechte von <button type="button" class="article-term" data-term-id="term-34">Eigentümern</button>. Außerdem sehen sie Probleme beim <button type="button" class="article-term" data-term-id="term-35">Datenschutz</button>. Einige <button type="button" class="article-term" data-term-id="term-36">zweifeln</button> auch, ob Berlin dafür <button type="button" class="article-term" data-term-id="term-37">überhaupt</button> eigene Regeln machen darf. Denn viele Fragen zu Mietpreisen regelt der Bund. <button type="button" class="article-term" data-term-id="term-38">Vor Gericht</button> wird wohl wichtig, welchen <button type="button" class="article-term" data-term-id="term-39">Zweck</button> das Register hat. Sammelt Berlin nur Daten zur Kontrolle <button type="button" class="article-term" data-term-id="term-40">bestehender</button> Regeln, könnte es <button type="button" class="article-term" data-term-id="term-41">erlaubt</button> sein. <button type="button" class="article-term" data-term-id="term-42">Schafft</button> das Land aber neue Preisregeln, wäre es schwieriger. Auch Wohnungsunternehmen nennen praktische und <button type="button" class="article-term article-term--default" data-term-id="term-43">rechtliche Hürden</button>, <button type="button" class="article-term" data-term-id="term-44">obwohl</button> sie mehr <button type="button" class="article-term" data-term-id="term-45">Marktübersicht</button> <button type="button" class="article-term" data-term-id="term-46">grundsätzlich</button> gut finden.

Am Ende <button type="button" class="article-term" data-term-id="term-47">entscheidet</button> nicht nur die <button type="button" class="article-term" data-term-id="term-48">Datensammlung</button> über den Erfolg. <button type="button" class="article-term" data-term-id="term-49">Bezirke</button> oder <button type="button" class="article-term" data-term-id="term-50">Staatsanwälte</button> müssten jeden <button type="button" class="article-term" data-term-id="term-51">Verdachtsfall</button> <button type="button" class="article-term" data-term-id="term-52">einzeln</button> <button type="button" class="article-term" data-term-id="term-13">prüfen</button>. Bei <button type="button" class="article-term" data-term-id="term-53">bestimmten</button> <button type="button" class="article-term article-term--default" data-term-id="term-15">Verstößen</button> müssen sie außerdem zeigen, dass Vermieter eine <button type="button" class="article-term article-term--default" data-term-id="term-54">persönliche Notlage</button> <button type="button" class="article-term" data-term-id="term-55">ausgenutzt</button> haben. Daran <button type="button" class="article-term" data-term-id="term-56">scheiterten</button> <button type="button" class="article-term" data-term-id="term-57">Verfahren</button> bisher oft, auch weil <button type="button" class="article-term" data-term-id="term-58">Personal fehlt</button>. Trotzdem schauen andere Städte mit teurem <button type="button" class="article-term" data-term-id="term-59">Wohnungsmarkt</button> genau nach Berlin. Das Register könnte ein neues <button type="button" class="article-term" data-term-id="term-60">Mittel</button> gegen <button type="button" class="article-term" data-term-id="term-61">Missbrauch</button> werden. Ein <button type="button" class="article-term article-term--default" data-term-id="term-62">wirklicher Durchbruch</button> kommt aber erst, wenn aus Daten <button type="button" class="article-term" data-term-id="term-63">wirksame</button> <button type="button" class="article-term" data-term-id="term-57">Verfahren</button> folgen.
