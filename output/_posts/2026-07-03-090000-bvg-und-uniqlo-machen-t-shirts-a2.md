Die BVG und Uniqlo starten eine <button type="button" class="article-term article-term--default" data-term-id="term-1">Modeaktion</button> in Berlin. Die BVG <button type="button" class="article-term" data-term-id="term-2">fährt</button> Busse, U-Bahnen und <button type="button" class="article-term" data-term-id="term-3">Straßenbahnen</button> in der Stadt. <button type="button" class="article-term" data-term-id="term-4">Seit</button> <button type="button" class="article-term" data-term-id="term-5">Donnerstag</button> können <button type="button" class="article-term" data-term-id="term-6">Kunden</button> T-Shirts und <button type="button" class="article-term" data-term-id="term-7">Taschen</button> <button type="button" class="article-term article-term--default" data-term-id="term-8">selbst gestalten</button>.

Das <button type="button" class="article-term" data-term-id="term-9">passiert</button> im großen Uniqlo-<button type="button" class="article-term" data-term-id="term-10">Geschäft</button> am Tauentzien in Berlin. <button type="button" class="article-term" data-term-id="term-11">Zuerst</button> <button type="button" class="article-term" data-term-id="term-12">wählen</button> die <button type="button" class="article-term" data-term-id="term-6">Kunden</button> ein <button type="button" class="article-term" data-term-id="term-13">Kleidungsstück</button> oder eine Tasche. Dann <button type="button" class="article-term" data-term-id="term-12">wählen</button> sie Bilder aus dem Berliner <button type="button" class="article-term article-term--default" data-term-id="term-14">Nahverkehr</button>. Als Bilder gibt es U-Bahn-<button type="button" class="article-term" data-term-id="term-15">Linien</button>, <button type="button" class="article-term article-term--default" data-term-id="term-16">Bahnhofsschilder</button> und BVG-<button type="button" class="article-term" data-term-id="term-17">Zeichen</button>. Die <button type="button" class="article-term" data-term-id="term-6">Kunden</button> können die Bilder <button type="button" class="article-term" data-term-id="term-18">frei</button> <button type="button" class="article-term article-term--default" data-term-id="term-19">verschieben</button> und Text <button type="button" class="article-term article-term--default" data-term-id="term-20">hinzufügen</button>.

Die Aktion soll <button type="button" class="article-term" data-term-id="term-21">bis Ende</button> des Jahres <button type="button" class="article-term" data-term-id="term-22">laufen</button>. Die <button type="button" class="article-term" data-term-id="term-23">Sachen</button> haben keinen <button type="button" class="article-term" data-term-id="term-24">Nutzen</button> für eine <button type="button" class="article-term" data-term-id="term-25">Fahrt</button>. Das war bei einem besonderen Adidas-Schuh im Jahr 2018 <button type="button" class="article-term" data-term-id="term-26">anders</button>. <button type="button" class="article-term" data-term-id="term-27">Damals</button> gab es nur 500 <button type="button" class="article-term" data-term-id="term-28">Paare</button> von diesem Schuh. Der Schuh hat auch als <button type="button" class="article-term article-term--default" data-term-id="term-29">Fahrkarte</button> für ein Jahr funktioniert.

Ein Shirt für <button type="button" class="article-term" data-term-id="term-30">Kinder</button> <button type="button" class="article-term" data-term-id="term-31">kostet ab</button> 12,90 Euro. Shirts für <button type="button" class="article-term" data-term-id="term-32">Erwachsene</button> und kleine <button type="button" class="article-term" data-term-id="term-7">Taschen</button> kosten 19,90 Euro. Eine große Tasche kostet 24,90 Euro im <button type="button" class="article-term" data-term-id="term-10">Geschäft</button>. <button type="button" class="article-term" data-term-id="term-33">Außerdem</button> gibt es <button type="button" class="article-term" data-term-id="term-34">fertige</button> Bilder für einen <button type="button" class="article-term" data-term-id="term-35">schnellen Kauf</button>. Uniqlo <button type="button" class="article-term" data-term-id="term-36">druckt</button> die Bilder direkt im <button type="button" class="article-term" data-term-id="term-10">Geschäft</button> auf <button type="button" class="article-term" data-term-id="term-37">Stoff</button>.

Die BVG <button type="button" class="article-term" data-term-id="term-38">erlaubt</button> viel bei den <button type="button" class="article-term" data-term-id="term-39">eigenen</button> <button type="button" class="article-term" data-term-id="term-17">Zeichen</button>. <button type="button" class="article-term" data-term-id="term-6">Kunden</button> <button type="button" class="article-term" data-term-id="term-40">dürfen</button> die Bilder <button type="button" class="article-term" data-term-id="term-41">ändern</button> und mit Text <button type="button" class="article-term" data-term-id="term-42">beschreiben</button>. Aber <button type="button" class="article-term" data-term-id="term-6">Kunden</button> <button type="button" class="article-term" data-term-id="term-40">dürfen</button> keine <button type="button" class="article-term article-term--default" data-term-id="term-43">verbotenen Inhalte</button> <button type="button" class="article-term" data-term-id="term-44">benutzen</button>. <button type="button" class="article-term" data-term-id="term-45">Früher</button> hat die BVG auch <button type="button" class="article-term" data-term-id="term-46">Sitzmuster</button> und Farben <button type="button" class="article-term" data-term-id="term-47">verkauft</button>. Ein <button type="button" class="article-term" data-term-id="term-48">Streit</button> um ein altes <button type="button" class="article-term" data-term-id="term-46">Sitzmuster</button> hat Probleme <button type="button" class="article-term" data-term-id="term-49">gezeigt</button>. <button type="button" class="article-term" data-term-id="term-50">Deshalb</button> nutzt die BVG jetzt <button type="button" class="article-term" data-term-id="term-51">lieber</button> <button type="button" class="article-term" data-term-id="term-52">klare Motive</button>.
