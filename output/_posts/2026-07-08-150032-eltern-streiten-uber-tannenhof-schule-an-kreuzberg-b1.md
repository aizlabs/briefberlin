An der Rosa-Parks-<button type="button" class="article-term" data-term-id="term-1">Grundschule</button> in Berlin-Kreuzberg gibt es <button type="button" class="article-term" data-term-id="term-2">Streit</button> um neue <button type="button" class="article-term" data-term-id="term-3">Räume</button> für die Tannenhof-Schule. Diese Schule <button type="button" class="article-term" data-term-id="term-4">unterstützt</button> <button type="button" class="article-term" data-term-id="term-5">Jugendliche</button> und <button type="button" class="article-term" data-term-id="term-6">Erwachsene</button>, die <button type="button" class="article-term" data-term-id="term-7">früher</button> <button type="button" class="article-term article-term--default" data-term-id="term-8">suchtkrank</button> waren und wieder lernen wollen. Sie soll für zwei Jahre in einen Teil des Grundschulgebäudes ziehen. Viele Eltern erfuhren <button type="button" class="article-term" data-term-id="term-9">nach eigenen Angaben</button> erst spät von den <button type="button" class="article-term" data-term-id="term-10">Plänen</button>.

Die <button type="button" class="article-term" data-term-id="term-11">Sorge</button> ist groß, weil Eltern, Kinder und <button type="button" class="article-term" data-term-id="term-12">Lehrkräfte</button> nicht <button type="button" class="article-term article-term--default" data-term-id="term-13">mitreden</button> konnten. Mehrere Eltern <button type="button" class="article-term" data-term-id="term-14">verlangen</button> klare Regeln für <button type="button" class="article-term" data-term-id="term-15">Schutz</button>, Kontakt und Information. Für Dienstag haben sie eine <button type="button" class="article-term article-term--default" data-term-id="term-16">Kundgebung</button> <button type="button" class="article-term" data-term-id="term-17">angemeldet</button>. <button type="button" class="article-term" data-term-id="term-18">Außerdem</button> läuft eine Petition, also eine <button type="button" class="article-term article-term--default" data-term-id="term-19">Unterschriftenliste</button>, für einen sicheren <button type="button" class="article-term" data-term-id="term-20">Lernort</button> für Kinder.

Die Tannenhof-Schule gibt es seit etwa 20 Jahren. Dort können Menschen mit früherer <button type="button" class="article-term" data-term-id="term-21">Sucht</button> einen höheren <button type="button" class="article-term article-term--default" data-term-id="term-22">Schulabschluss</button> <button type="button" class="article-term" data-term-id="term-23">nachholen</button>, sogar das <button type="button" class="article-term" data-term-id="term-24">Abitur</button>. Wer dort lernt, muss <button type="button" class="article-term" data-term-id="term-25">ohne Alkohol</button> und andere Drogen leben. Die Schule kontrolliert diese Abstinenz regelmäßig. <button type="button" class="article-term" data-term-id="term-26">Zuletzt</button> fand sie aber keine <button type="button" class="article-term" data-term-id="term-27">passenden</button> <button type="button" class="article-term" data-term-id="term-3">Räume</button>. Schon in Neukölln gab es <button type="button" class="article-term" data-term-id="term-28">Proteste</button> gegen einen <button type="button" class="article-term" data-term-id="term-29">Umzug</button> an eine <button type="button" class="article-term" data-term-id="term-1">Grundschule</button>. <button type="button" class="article-term" data-term-id="term-30">Deshalb</button> soll die Schule nun nach Kreuzberg kommen.

Der <button type="button" class="article-term" data-term-id="term-31">Bezirk</button> und die Berliner Schulbehörde planen einen <button type="button" class="article-term article-term--default" data-term-id="term-32">getrennten Zugang</button> über die Forster Straße. Die Kinder der <button type="button" class="article-term" data-term-id="term-1">Grundschule</button> kommen weiter über die Reichenberger Straße <button type="button" class="article-term" data-term-id="term-33">hinein</button>. So wollen die <button type="button" class="article-term" data-term-id="term-34">Behörden</button> Treffen zwischen den kleinen Kindern und den Tannenhof-Lernenden <button type="button" class="article-term" data-term-id="term-35">möglichst</button> <button type="button" class="article-term" data-term-id="term-36">vermeiden</button>. Die <button type="button" class="article-term" data-term-id="term-37">Nutzung</button> ist <button type="button" class="article-term" data-term-id="term-38">begrenzt</button>, denn die <button type="button" class="article-term" data-term-id="term-1">Grundschule</button> braucht die <button type="button" class="article-term" data-term-id="term-3">Räume</button> <button type="button" class="article-term" data-term-id="term-39">später</button> selbst. <button type="button" class="article-term" data-term-id="term-40">Grund dafür</button> ist eine <button type="button" class="article-term" data-term-id="term-41">geplante</button> <button type="button" class="article-term article-term--default" data-term-id="term-42">Sanierung</button>, also eine größere Reparatur des <button type="button" class="article-term" data-term-id="term-43">Gebäudes</button>. Nach den <button type="button" class="article-term" data-term-id="term-44">Sommerferien</button> wollen die <button type="button" class="article-term" data-term-id="term-45">Verantwortlichen</button> weitere <button type="button" class="article-term" data-term-id="term-46">Einzelheiten</button> <button type="button" class="article-term" data-term-id="term-47">klären</button>.

Unter den Eltern gibt es <button type="button" class="article-term" data-term-id="term-48">unterschiedliche Meinungen</button>. <button type="button" class="article-term" data-term-id="term-49">Kritiker</button> <button type="button" class="article-term" data-term-id="term-50">verweisen auf</button> Probleme im <button type="button" class="article-term" data-term-id="term-51">Umfeld</button> des Görlitzer Parks und <button type="button" class="article-term" data-term-id="term-52">fürchten</button> mehr <button type="button" class="article-term" data-term-id="term-53">Belastung</button>. Andere <button type="button" class="article-term" data-term-id="term-54">warnen</button> <button type="button" class="article-term" data-term-id="term-55">jedoch</button> <button type="button" class="article-term" data-term-id="term-56">davor</button>, die Tannenhof-Lernenden wegen ihrer <button type="button" class="article-term" data-term-id="term-57">Vergangenheit</button> schlecht zu <button type="button" class="article-term" data-term-id="term-58">behandeln</button>. Sie sehen auch eine Chance für ein anderes Bild nach einer <button type="button" class="article-term article-term--default" data-term-id="term-59">Suchterkrankung</button>. Die <button type="button" class="article-term" data-term-id="term-60">Schulleitung</button> der <button type="button" class="article-term" data-term-id="term-1">Grundschule</button> ist <button type="button" class="article-term" data-term-id="term-61">grundsätzlich</button> <button type="button" class="article-term" data-term-id="term-62">zuversichtlich</button>. <button type="button" class="article-term" data-term-id="term-63">Trotzdem</button> zeigen erst weitere <button type="button" class="article-term" data-term-id="term-64">Gespräche</button>, ob die <button type="button" class="article-term" data-term-id="term-65">Sorgen</button> der Eltern kleiner werden.
