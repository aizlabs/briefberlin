Teslas Fabrik in Grünheide bei Berlin hat 2025 mehr <button type="button" class="article-term article-term--default" data-term-id="term-1">Gewinn</button> gemacht. <button type="button" class="article-term" data-term-id="term-2">Das passierte</button>, <button type="button" class="article-term" data-term-id="term-3">obwohl</button> Tesla <button type="button" class="article-term" data-term-id="term-4">weltweit</button> weniger Autos <button type="button" class="article-term" data-term-id="term-5">verkaufte</button> und in Brandenburg weniger <button type="button" class="article-term" data-term-id="term-6">Fahrzeuge</button> <button type="button" class="article-term" data-term-id="term-7">baute</button>. Die deutsche <button type="button" class="article-term" data-term-id="term-8">Produktionsfirma</button> <button type="button" class="article-term" data-term-id="term-9">verdiente</button> 77,1 Millionen Euro. Im Jahr 2024 waren es 56,8 Millionen Euro. Der <button type="button" class="article-term article-term--default" data-term-id="term-10">Umsatz</button>, also das Geld <button type="button" class="article-term" data-term-id="term-11">aus Verkäufen</button>, <button type="button" class="article-term article-term--default" data-term-id="term-12">sank</button> <button type="button" class="article-term" data-term-id="term-13">jedoch</button> von 7,6 auf 7,1 Milliarden Euro. Ein wichtiger Grund ist: Tesla <button type="button" class="article-term" data-term-id="term-14">senkte</button> die <button type="button" class="article-term" data-term-id="term-15">Kosten</button> in der Produktion <button type="button" class="article-term" data-term-id="term-16">weiter</button>.

In Grünheide liefen 2025 <button type="button" class="article-term" data-term-id="term-17">rund</button> 202.000 <button type="button" class="article-term" data-term-id="term-6">Fahrzeuge</button> <button type="button" class="article-term article-term--default" data-term-id="term-18">vom Band</button>. Das waren etwa 9.000 weniger als im <button type="button" class="article-term" data-term-id="term-19">Vorjahr</button>. Die Fabrik nutzte 54 Prozent ihrer möglichen <button type="button" class="article-term" data-term-id="term-20">Leistung</button>. <button type="button" class="article-term" data-term-id="term-21">Damit</button> <button type="button" class="article-term" data-term-id="term-22">lag</button> sie <button type="button" class="article-term" data-term-id="term-23">leicht</button> unter dem <button type="button" class="article-term" data-term-id="term-24">Wert</button> von 2024. <button type="button" class="article-term" data-term-id="term-25">Außerdem</button> <button type="button" class="article-term" data-term-id="term-7">baute</button> Tesla einige <button type="button" class="article-term" data-term-id="term-26">Produktionslinien</button> um. <button type="button" class="article-term" data-term-id="term-27">Dabei</button> <button type="button" class="article-term" data-term-id="term-28">ging es um</button> neue Varianten des Model Y und um eine Version für Kanada. <button type="button" class="article-term" data-term-id="term-29">Seit</button> dem Start der Produktion 2022 hat Tesla in Brandenburg <button type="button" class="article-term" data-term-id="term-17">rund</button> 700.000 <button type="button" class="article-term" data-term-id="term-6">Fahrzeuge</button> gebaut. Das Model Y aus Grünheide <button type="button" class="article-term" data-term-id="term-30">blieb</button> 2025 das <button type="button" class="article-term" data-term-id="term-31">meistverkaufte</button> Elektroauto in Europa. Heute <button type="button" class="article-term" data-term-id="term-33">liefert</button> das <button type="button" class="article-term" data-term-id="term-32">Werk</button> Autos in mehr als 30 <button type="button" class="article-term" data-term-id="term-34">Märkte</button>.

Für die <button type="button" class="article-term" data-term-id="term-35">nächsten Jahre</button> plant Tesla in Grünheide starkes <button type="button" class="article-term" data-term-id="term-36">Wachstum</button>. Die <button type="button" class="article-term article-term--default" data-term-id="term-37">Behörden</button> <button type="button" class="article-term" data-term-id="term-38">erlauben</button> dort <button type="button" class="article-term" data-term-id="term-39">bis zu</button> 500.000 <button type="button" class="article-term" data-term-id="term-6">Fahrzeuge</button> pro Jahr. Technisch kann das <button type="button" class="article-term" data-term-id="term-32">Werk</button> <button type="button" class="article-term" data-term-id="term-40">zurzeit</button> <button type="button" class="article-term" data-term-id="term-41">wohl</button> etwa 375.000 Autos <button type="button" class="article-term" data-term-id="term-42">schaffen</button>. Dieses <button type="button" class="article-term" data-term-id="term-43">Niveau</button> will Tesla <button type="button" class="article-term" data-term-id="term-44">möglichst schnell</button> <button type="button" class="article-term" data-term-id="term-45">erreichen</button>. <button type="button" class="article-term" data-term-id="term-46">Dafür</button> <button type="button" class="article-term" data-term-id="term-47">müsste</button> die Fabrik <button type="button" class="article-term" data-term-id="term-48">ungefähr</button> 7.500 Autos pro Woche bauen. <button type="button" class="article-term" data-term-id="term-49">Gleichzeitig</button> arbeitet Tesla an einer kompletten Produktion von <button type="button" class="article-term article-term--default" data-term-id="term-50">Batteriezellen</button>. Diese Zellen sind wichtige <button type="button" class="article-term" data-term-id="term-51">Teile</button> einer <button type="button" class="article-term" data-term-id="term-52">Autobatterie</button>. Der Start ist für 2027 <button type="button" class="article-term" data-term-id="term-53">geplant</button>. Tesla will <button type="button" class="article-term" data-term-id="term-46">dafür</button> <button type="button" class="article-term" data-term-id="term-17">rund</button> eine Milliarde Euro <button type="button" class="article-term" data-term-id="term-54">investieren</button>. Die geplante <button type="button" class="article-term" data-term-id="term-20">Leistung</button> liegt bei acht Gigawattstunden pro Jahr. Diese Zahl zeigt, wie viel Energie die Batterien <button type="button" class="article-term" data-term-id="term-55">speichern</button> können. <button type="button" class="article-term" data-term-id="term-56">Ende 2025</button> arbeiteten <button type="button" class="article-term" data-term-id="term-57">gut 11.000</button> Menschen im <button type="button" class="article-term" data-term-id="term-32">Werk</button>. Nun <button type="button" class="article-term" data-term-id="term-58">kündigt</button> Tesla weitere <button type="button" class="article-term article-term--default" data-term-id="term-59">feste Stellen</button> an, besonders für <button type="button" class="article-term article-term--default" data-term-id="term-60">Fachkräfte</button> in der Batterieproduktion.

Diese Pläne <button type="button" class="article-term" data-term-id="term-61">unterscheiden</button> Tesla von vielen großen deutschen <button type="button" class="article-term" data-term-id="term-62">Autoherstellern</button>. Viele von ihnen <button type="button" class="article-term article-term--default" data-term-id="term-63">bauen Stellen ab</button> oder <button type="button" class="article-term" data-term-id="term-64">prüfen</button> <button type="button" class="article-term" data-term-id="term-65">einzelne Werke</button>. Gründe sind <button type="button" class="article-term" data-term-id="term-67">schwächere</button> <button type="button" class="article-term" data-term-id="term-66">Nachfrage</button>, hohe <button type="button" class="article-term" data-term-id="term-15">Kosten</button> und verschiedene Pläne für Elektro-, Hybrid- und <button type="button" class="article-term" data-term-id="term-68">Verbrennerautos</button>. Tesla setzt <button type="button" class="article-term" data-term-id="term-69">dagegen</button> klar auf Elektroautos und eine <button type="button" class="article-term" data-term-id="term-70">einheitliche</button> technische Basis. <button type="button" class="article-term" data-term-id="term-71">Trotzdem</button> bleiben Risiken. <button type="button" class="article-term" data-term-id="term-72">Lieferprobleme</button>, politische <button type="button" class="article-term" data-term-id="term-73">Spannungen</button> in der Welt, Sabotage und ein schlechtes <button type="button" class="article-term" data-term-id="term-74">öffentliches Bild</button> könnten <button type="button" class="article-term" data-term-id="term-75">schaden</button>. Auch die starke <button type="button" class="article-term" data-term-id="term-76">Aufmerksamkeit</button> für Tesla und seine <button type="button" class="article-term" data-term-id="term-77">Führung</button> kann Verkäufe <button type="button" class="article-term" data-term-id="term-78">belasten</button>.

## Vokabeln

- **Gewinn** - profit - Geld, das nach Abzug der Kosten übrig bleibt.
- **Umsatz** - revenue, sales - Geld, das eine Firma durch Verkäufe einnimmt.
- **sank** - fell, decreased - Wurde niedriger oder weniger.
- **vom Band** - off the production line - Aus einer Fabrikproduktion heraus, fertig hergestellt.
- **Behörden** - authorities - Staatliche Stellen oder Ämter mit Entscheidungsrecht.
- **Batteriezellen** - battery cells - Kleine Einheiten, die elektrische Energie in Batterien speichern.
- **feste Stellen** - permanent positions - Arbeitsplätze ohne kurze Befristung.
- **Fachkräfte** - skilled workers, specialists - Menschen mit besonderer Ausbildung oder Fachwissen.
- **bauen Stellen ab** - cut jobs - Arbeitsplätze reduzieren oder streichen.
