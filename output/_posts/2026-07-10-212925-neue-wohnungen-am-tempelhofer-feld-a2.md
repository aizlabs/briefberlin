Das ist eine große <button type="button" class="article-term article-term--default" data-term-id="term-3">freie Fläche</button> in Berlin. Eine neue <button type="button" class="article-term article-term--default" data-term-id="term-4">Rechnung</button> zeigt jetzt <button type="button" class="article-term" data-term-id="term-5">Kosten</button> und <button type="button" class="article-term" data-term-id="term-6">mögliche</button> <button type="button" class="article-term" data-term-id="term-7">Mieten</button>.

<button type="button" class="article-term article-term--default" data-term-id="term-8">Am Rand</button> des Feldes sollen vielleicht viele <button type="button" class="article-term article-term--default" data-term-id="term-1">Wohnungen</button> <button type="button" class="article-term article-term--default" data-term-id="term-9">entstehen</button>. Die <button type="button" class="article-term article-term--default" data-term-id="term-4">Rechnung</button> sagt: Der <button type="button" class="article-term" data-term-id="term-10">Bau</button> kann <button type="button" class="article-term" data-term-id="term-11">finanziell funktionieren</button>. <button type="button" class="article-term" data-term-id="term-12">Das heißt</button>: Die <button type="button" class="article-term" data-term-id="term-5">Kosten</button> können <button type="button" class="article-term" data-term-id="term-13">zum Plan passen</button>. Aber die Stadt hat noch nicht <button type="button" class="article-term" data-term-id="term-14">Ja gesagt</button>.

An der <button type="button" class="article-term" data-term-id="term-15">Kante</button> des Feldes plant Berlin ein neues <button type="button" class="article-term article-term--default" data-term-id="term-16">Wohngebiet</button>. Dort sollen <button type="button" class="article-term" data-term-id="term-17">mehr als</button> 21.000 <button type="button" class="article-term article-term--default" data-term-id="term-1">Wohnungen</button> für <button type="button" class="article-term" data-term-id="term-18">etwa</button> 50.000 Menschen <button type="button" class="article-term article-term--default" data-term-id="term-9">entstehen</button>. Damit <button type="button" class="article-term" data-term-id="term-19">gehört</button> der Plan zu Berlins <button type="button" class="article-term" data-term-id="term-20">größten</button> Plänen für <button type="button" class="article-term article-term--default" data-term-id="term-1">Wohnungen</button>. <button type="button" class="article-term" data-term-id="term-21">Zuerst</button> gibt es aber nur die <button type="button" class="article-term article-term--default" data-term-id="term-4">Rechnung</button> und Ideen.

Die neuen Häuser sollen <button type="button" class="article-term" data-term-id="term-22">nah</button> <button type="button" class="article-term article-term--default" data-term-id="term-8">am Rand</button> stehen. In der <button type="button" class="article-term" data-term-id="term-23">Mitte</button> <button type="button" class="article-term" data-term-id="term-24">bleibt</button> das große Feld <button type="button" class="article-term" data-term-id="term-25">weiter</button> frei. Dieser freie <button type="button" class="article-term" data-term-id="term-26">Teil</button> ist <button type="button" class="article-term" data-term-id="term-18">etwa</button> 200 Hektar groß. Das ist sehr viel <button type="button" class="article-term" data-term-id="term-27">Platz</button> für Sport, <button type="button" class="article-term" data-term-id="term-28">Wege</button> und Natur.

Die Häuser sollen <button type="button" class="article-term" data-term-id="term-29">ähnlich</button> wie in <button type="button" class="article-term" data-term-id="term-30">nahen</button> <button type="button" class="article-term" data-term-id="term-31">Stadtteilen</button> <button type="button" class="article-term" data-term-id="term-32">aussehen</button>. Dort stehen viele Häuser <button type="button" class="article-term" data-term-id="term-33">dicht</button> in festen <button type="button" class="article-term" data-term-id="term-34">Blöcken</button>. Die <button type="button" class="article-term" data-term-id="term-35">Firmen</button> sollen die neuen Häuser schnell nach ähnlichen Plänen bauen. <button type="button" class="article-term" data-term-id="term-36">Dafür</button> <button type="button" class="article-term" data-term-id="term-37">nutzen</button> sie oft <button type="button" class="article-term" data-term-id="term-38">gleiche</button> Teile und <button type="button" class="article-term" data-term-id="term-38">gleiche</button> Pläne.

<button type="button" class="article-term" data-term-id="term-39">Außerdem</button> sprechen Menschen über den <button type="button" class="article-term" data-term-id="term-40">Schutz</button> der freien Fläche. Eine Organisation <button type="button" class="article-term article-term--default" data-term-id="term-41">ohne Gewinnziel</button> kann die <button type="button" class="article-term article-term--default" data-term-id="term-3">freie Fläche</button> <button type="button" class="article-term" data-term-id="term-42">sichern</button>. Dann <button type="button" class="article-term" data-term-id="term-24">bleibt</button> der große <button type="button" class="article-term" data-term-id="term-43">Mittelteil</button> <button type="button" class="article-term" data-term-id="term-44">dauerhaft</button> frei. <button type="button" class="article-term" data-term-id="term-45">Am Ende</button> müssen Berliner Politiker über den <button type="button" class="article-term" data-term-id="term-10">Bau</button> <button type="button" class="article-term article-term--default" data-term-id="term-46">entscheiden</button>.
